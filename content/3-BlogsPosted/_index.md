This section provides an overview of the blog posts I have written and published during my exploration of AWS services. The articles focus on Cloud Architecture, Serverless Computing, Generative AI, and AI Agents, while sharing practical knowledge and demonstrating how AWS services can be combined to solve real-world problems.

### [Blog 1 - Building a Microservices Architecture with Amazon SQS, Amazon SNS, and AWS Lambda](3.1-Blog1/)

This article introduces how to build a microservices architecture using asynchronous communication by combining Amazon SQS, Amazon SNS, and AWS Lambda. It explains a real-world business scenario, the overall system architecture, the processing workflow, and the role of each AWS service. The article also highlights the benefits of Event-Driven and Serverless architectures in reducing service dependencies, improving scalability, and increasing system fault tolerance.

### [Blog 2 - Building an AI AWS Architecture Reviewer with Amazon Bedrock and Serverless](3.2-Blog2/)

This article presents the design of the AI AWS Architecture Reviewer, a Generative AI solution built on AWS that automatically analyzes AWS architecture diagrams, evaluates them based on the AWS Well-Architected Framework, estimates deployment costs, and generates PDF reports. It explains the Serverless and Event-Driven architecture, the system workflow, and the responsibilities of key AWS services such as Amazon Bedrock, AWS Lambda, Amazon EventBridge, AWS Step Functions, Amazon S3, and Amazon DynamoDB. The article also discusses why this architecture was chosen for the project.

### [Blog 3 - Amazon Bedrock AgentCore Web Search: When AI Agents Are No Longer Limited by Knowledge Cutoff](3.3-Blog3/)

This article introduces Web Search on Amazon Bedrock AgentCore, a feature that enables AI Agents to retrieve up-to-date information from the web and generate responses that are grounded with reliable sources. It discusses the limitations of AI Agents that rely solely on static knowledge, explains the architecture of Web Search, and describes the roles of AgentCore Gateway and the Model Context Protocol (MCP). In addition, the article explores practical use cases and the benefits of integrating managed web search capabilities into enterprise AI Agent solutions.
