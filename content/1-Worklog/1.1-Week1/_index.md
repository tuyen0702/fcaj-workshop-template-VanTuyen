### Lab implementation steps for week 1:

* Create an AWS account.
* Set up a virtual MFA device.
* Create an admin group and an admin user.
* Create a Budget by Template.
* Create a Cost Budget tutorial.
* Review AWS Support Packages.

### Results achieved in week 1:

* Understood that AWS is a cloud platform that provides many service groups such as Compute, Storage, Networking, Database, Security, and Monitoring.
* Understood Region, Availability Zone, Edge Location, and the role of AWS global infrastructure.
* Became familiar with AWS Management Console and knew how to search for services.
* Understood the role of MFA, IAM User, IAM Group, IAM Role, and IAM Policy.
* Understood how to use AWS Budget to control costs during practice.
