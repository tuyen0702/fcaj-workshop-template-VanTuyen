### Các bước thực hiện lab trong tuần 1:

* Create an AWS account.
* Setup with Virtual MFA Device.
* Create admin group and admin user.
* Create Budget by Template.
* Create Cost Budget Tutorial.
* Review AWS Support Packages.

### Kết quả đạt được tuần 1:

* Hiểu được AWS là nền tảng cloud cung cấp nhiều nhóm dịch vụ như Compute, Storage, Networking, Database, Security, Monitoring.
* Nắm được khái niệm Region, Availability Zone, Edge Location và vai trò của hạ tầng toàn cầu AWS.
* Làm quen với AWS Management Console và biết cách tìm kiếm dịch vụ.
* Hiểu vai trò của MFA, IAM User, IAM Group, IAM Role và IAM Policy.
* Nắm được cách dùng AWS Budget để kiểm soát chi phí trong quá trình thực hành.
